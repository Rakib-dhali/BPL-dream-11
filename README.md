# 🏏 BPL Dream 11

A fantasy cricket team builder inspired by the Bangladesh Premier League (BPL). Users can browse available players, manage their squad by selecting or removing players, and track their remaining coin budget — all in a clean, responsive UI.

## 🔗 Live Demo

[View Live](https://bpl-dream-11-rakib.netlify.app/) 

## ✨ Features

- Browse a list of BPL cricketers with stats and details
- Select players to build your dream team (max 11 players)
- Remove players from your selected squad
- Coin-based budget system — spend wisely!
- Toast notifications for user feedback (player added/removed/errors)
- Fully responsive design with Tailwind CSS + DaisyUI
- Tab-based navigation between "Available Players" and "Selected Players"

## 🛠️ Tech Stack

- **React 19** — UI library
- **Vite 7** — Build tool & dev server
- **Tailwind CSS v4** — Utility-first styling
- **DaisyUI v5** — Tailwind component library
- **React Icons** — Icon pack
- **React Toastify** — Toast notification system

## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- npm

### Installation
```bash
# Clone the repository
git clone https://github.com/Rakib-dhali/BPL-dream-11.git

# Navigate into the project
cd BPL-dream-11

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production
```bash
npm run build
```
